- React is an external library.
  ● An external library is a bunch of code that is outside of our computer.
  ● It's code that someone else wrote and we can load that code into our website.
- React helps us create websites easier



- To create websites
  ● React : contains all the shared features that are used in both websites and mobile apps
  ● React DOM : contain features that are specific to websites
- To create mobile apps
  ● React : contains all the shared features that are used in both websites and mobile apps
  ● React Native : contain features that are specific to mobile apps



- import React in website :

  <script src="https://unpkg.com/supersimpledev/react.js"> </script>
  <script src="https://unpkg.com/supersimpledev/react-dom.js"> </script>  



  ● in React we don't use normal JavaScript.
  ● We use an enhanced version of JavaScript called jsx (JavaScript XML).
  ● jsx : It's the same as JavaScript, but we can also write HTML directly in our JavaScript code.
  ● browser can understand jsx code
  ● we use Babel(external .lip) to translates other languages into js.
  ● Babel like JavaScript compiler

- How to use React :
  ● create container (div)
  ● use jsx to display all pages in that container

- How to write :
  ● call Babel
  ● create script element with attribute type equals text/babel.
  ● write jsx code
  ● create container
  ● display container
 
<script src="https://unpkg.com/supersimpledev/babel.js"> </script>
<script type="text/babel">
// jsx code
// container
</script>
