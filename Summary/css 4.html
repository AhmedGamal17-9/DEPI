Media Type: نوع الوسط اللي بيتعرض فيه المحتوى
all (default) → الكل 
screen → خاص بالشاشات (كمبيوتر، لابتوب، موبايل، تابلت)
print → بيتطبق فقط لما المستخدم يطبع الصفحة
speech → للمستعرضات اللي تقرأ الصفحة بصوت

only :
- زمان قبل CSS3
- في متصفحات قديمة جدًا ماكنتش بتفهم الميديا كويري
- فكانوا ساعات يطبقوا القاعدة غلط أو يتجاهلوها بطريقة مش مظبوطة
- عشان كده اخترعوا كلمة only
- المتصفح الحديث يفهمها ويتجاهلها (يعاملها كأنها مش موجودة)
- المتصفح القديم مايعرفش الكلمة دي يرمي الميديا كويري كله ويتجاهله

Media Features : الشروط
1- max-width/min-width/min-height/max-height
2- px <= width <= px زي الرياضة

ربط الشروط :
and :  لازم كل الشروط تتحقق
, : طبق لو أي واحد فيهم اتحقق
not : يتطبق على أي حاجة غير المتحدد

animation: [name] [duration] [timing-function] [delay] [numper of iteration] [direction] [fill-mode];

❁name → اسم الحركة

❁duration → المدة

timing-function → السرعة        

delay → تأخير قبل البداية       

numper of iteration → عدد مرات التكرار (number or infinite) 

direction → اتجاه تشغيل الأنيميشن
  normal : (حركته فقط)يمشي من البداية الي النهاية
  alternate : يروح من البداية للنهاية، وبعدين يرجع بالعكس
  reverse : يمشي من النهاية الي البداية

  fill-mode → حالة العنصر قبل وبعد الأنيميشن
  none : بدون حالة
  backwards: قبل ما يبدأ : بياخد شكل العنصر في  خلال وقت الـديلاي 
  forwards : بعد ما يخلص :العنصر يقف مكانه الجديد، مايرجعش
  both :  يطبق forwards و backwards مع بعض
-----------------------
overflow:; بتتحكم إزاي المحتوى اللي جوا العنصر يتصرف لو زاد عن المساحة المحددة ليه

visible (default) → المحتوى بيبان حتى لو طالع برا الصندوق
hidden → المحتوى الزيادة يتقص وما يبانش
scroll → يظهر سكرول بار دايمًا حتى لو مش محتاج
auto → يظهر سكرول بار بس لو المحتوى زاد فعلًا
-----------------------
float: ; ✖️
1- بيخلي العنصر "يطير" ناحية اليمين أو الشمال
2- النصوص أو العناصر اللي بعده بتلتف حواليه (كأنها بتملأ الفراغ جنبه)
3- استخدام الفلوت كان زمان عشان نعمل (أعمدة وصفوف)
