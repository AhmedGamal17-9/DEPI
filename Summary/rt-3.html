- useState 
  ● store valus to change it after update
  ● state_initial_value => like constractor in life cycle method
  ● set state() => like setter in update in life cycle method
  ● const [state_name, method_changing_it] = useState(state_initial_value);
  ● It's forbidden to change the state directly (❌ count = 5).
  ● Correct ✅: setCount(5).
  
import { useState } from 'react';

function Counter(){
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>العداد: {count}</p>
      <button onClick={() => setCount(count + 1)}> + </button>
      <button onClick={() => setCount(count - 1)}> - </button>
      <button onClick={() => setCount(0)}> Reset to zero </button>
    </div>
  );
}

      ✨ spread operator :
 [ ...array-name ] 		
      
        - 🆕 It creates a new copy of the array or object 

        - ⚠️ in object If you write a property that already exists after or before the ... the last value overwrite the other-one. 
  
        - ⚠️ in object we can delete by spread operator : Destructuring . 
        
        - ℹ️ Not a function, just a syntax. 

        - usage: ✏️ To add/edit safely. 

        - usage: used with methods like filter or slice to give a new version of state in React. 

      		
    

    

      ✂️ .slice(start, end)
      
        - 🆕 Returns a new copy of array From (start) Until before (end) 

        - If there is no (end), then until length 

        - If there is no (start,end), then 📋 copy all array 

        - If Negative (-x) , then start from before (x) from the end 

        - usage: 🗑️ Delete item from Array. 

        - usage: 📋 Copy a of the All Array. 

        - usage: 📋 Copy a specific part of the Array. 

      		
    

    

      🔁 .map(
(element,index,array) => ()
)		
      
        - It iterates over each element in the array and applies the function you give it. 

        - Returns a new array containing the results with the same length. 

        - use key as a unique identifier for each element returned from .map() so React can track which items changed, added, or removed.
        - usage: 🔂 as a loop. 

        - usage: 🎨 To render elements in return(). 

        - usage: ✏️ To update Array. 

      		
    
  
    

      🗑️ .filter(
(element,index,array) => ()
)		
      
        - Creates a new array with all elements that pass the test. 

        - The condition inside must return true or false.
        - If true, the element remains in the array.
        - If false, the element is removed.
        - If no elements pass the test, it returns an empty array. 

        - usage: 🗑️ To Delete item from Array. 

        - usage: 🔍 Search/find in Array.

    🔍 find(
(element,index,array) => ()
)		
    
      - 🔍 Returns the value of the first element that passes a test. 

      - If no elements pass the test, it returns undefined. 

      - usage: 🔍 find one item in Array.

    		
  

  

    ✅❌ includes(value,start)
    
      - ✅ Checks if a certain element is present in the array or not. 

      - Uses === strict equality ⚖️ 

      - Returns ✅ true / ❌ false 

      - (start) is optional , default = 0 

      - ⚡ works with NaN
