Array Methods :

Method Description
push("") Add element at the end of array
pop() Delete the last item
unshift("") Add element to the beginning of the array
shift() Delete the first element
length Number of array elements
forEach() - Executes a function on each element
- Always goes through all elements
- Can't use break/continue
slice(start, end) - Does not change the original array
- Returns a new copy of array From (start) Until before (end)
- If there is no (end), then until length
- If there is no (start,end), then copy all array
- If Negative (-x) , then start from before (x) from the end
splice(start, deleteCount, item1, item2, ...) - Changes the original array
- Returns an array of deleted elements only
- Adds or removes elements anywhere in the array
- Deletes + adds at the same time
- From (start) delete (deleteCount) elements Then add (item1, item2, ...) from (start)it self
- If (deleteCount) = 0 , then no deletion
- If (item1, item2, ...) not exist , then no addition
- If there is no (deleteCount) , then delete all elements after the (start)
- If Negative (-x) , then start from before (x) from the end
includes(value,start) - Checks if a certain element is present in the array or not.
- Uses strict equality (===)
- Returns true/false
- (start) is optional , default = 0
- works with NaN
indexOf(value,start) - Searches for a specific element in the array.
- Search from beginning to end
- It returns the index of the first time it finds it.
- If it does not find it, it returns -1.
- (start) is optional , default = 0
- Uses strict equality (===)
- Does not work with NaN
lastIndexOf(value,start) - Search from end to beginning
- returns the index of the last time it finds it.
- The rest is the same as indexOf
find(function(){}) - Returns the value of the first element that passes a test.
- If no elements pass the test, it returns undefined.
- The test is a function that you provide.
filter - Creates a new array with all elements that pass the test.
- If no elements pass the test, it returns an empty array.
- The test is a function that you provide. 
notes: 
  - When you declare an array with const, you can change the elements inside the array,
    but you cannot reassign the array itself to a new value.
