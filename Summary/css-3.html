::pseudo-element  تستخدم علشان نحدد جزء من العنصر نفسه ونتعامل معاه كأنه عنصر منفصل
مثال :
::first-line ;
1- بيتحكم في شكل أول سطر من النص
2- ✅ color, font (size, weight, style, variant), word-spacing, letter-spacing, line-height, text-decoration, text-transform, background, vertical-align.
3- ❌ margin, padding, border, width, height, display, position, float, overflow, transform, box-shadow.

::first-letter ; 
1- شكل أول حرف من النص
2- ✅ color, font (size, weight, style, variant), word-spacing, letter-spacing, line-height, text-decoration, text-transform, text-shadow, background, margin, padding, border.
3- ❌ width, height, display, position, overflow, clear, transform.

::after{
  content: ""; لازم تكتب
}
::before ; {
  content: ""; لازم تكتب
}

:pseudo-class  تستخدم علشان نحدد حالة معينة للعنصر أو وضعه بالنسبة للـ DOM
مثال : 
hover,active,focus,link,visited,
first-child , last-child , nth-child()

display:grid; نظام يسمحلك تعمل تقسم العنصر صفوف و أعمدة

fr: جزء نسبي من المساحة المتاحة

grid-template-columns: 20%  20% 20%; 
grid-template-rows: 100px 200px auto;
grid-template-columns: repeat(3, 1fr);

height:100%;
grid-template-rows: minmax(minimum size,maximum size) 200px minmax(minimum size,maximum size);

grid-column-gap: 10px; مسافة بين الأعمدة
grid-row-gap: 20px; مسافة بين الصفوف

gap: 1fr 1fr;

align-items:; محاذاة رأسية ( بيتحكم في العناصر نفسها)

align-content: ; محاذاة رأسية ( بيتحكم في الصفوف الي بين العناصر)

justify-content:; محاذاة أفقية 
تحديد مكان العنصر
grid-column: grid-column-start / grid-column-end ; القيم هنا رقم العمود
grid-row: grid-row-start / grid-row-end ; 
grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end ;


grid-template-areas:  تسمحلك ترسم layout بالكلمات
    "header header"
    "main main"
    "footer footer" ;
.header { grid-area: header; }
.main { grid-area: main; }
.footer { grid-area: footer; }
