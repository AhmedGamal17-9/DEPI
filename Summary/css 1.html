  text-align: ; تحدد محاذاة النصوص داخل العنصر
  text-align:left; لو الموقع لغة واحدة و مش هيترجم : محاذاة النصوص لليسار (الافتراضي في اللغات اللاتينية) 
  text-align:right; لو الموقع لغة واحدة و مش هيترجم : محاذاة النصوص لليمين (الافتراضي في اللغات العربية)
  text-align:start; يحاذي النص حسب اتجاه اللغة (لو النص عربي → يمين، لو إنجليزي → يسار)
  text-align:end; العكس من start (لو عربي → يسار، لو إنجليزي → يمين) 
  text-align:center; النصوص تكون في منتصف السطر
  text-align: justify; توزيع النصوص على السطر بالكامل بحيث تكون محاذية لليمين واليسار (زي الجرائد)


  text-transform: ; تتحكم في شكل الحروف
  text-transform:none; النص يظهر زي ما هو
  text-transform:uppercase; كل الحروف الانجليزي كابيتال
  text-transform:lowercase; كل الحروف الانجليزي سمال
  text-transform:capitalize; أول حرف من كل كلمة كابيتال

  text-indent:50px ; مسافة قبل بداية السطر الأول فقط و مش بيشتغل علي الاسبان*

    9px 9px 0 #0077ff;

display: بتحدد إزاي العنصر يظهر على الصفحة
-----------------------
display: none
العنصر يختفي تمامًا (مش موجود في الصفحة ولا ياخد مساحة).
-----------------------
display: block 
العنصر ياخد العرض كامل
يبدأ في سطر جديد.
 تقدر تغيّر أبعاده (width / height).
-----------------------
display: inline
العنصر يظهر في نفس السطر بجانب العناصر الأخرى.
ياخد مساحة المحتوى فقط (مش العرض كامل).
ما تقدرش تغيّر أبعاده (width / height).
-----------------------
display: inline-block
خليط بين block و inline:
  زي inline: يجي في نفس السطر جنب غيره.
  زي block: تقدر تعدل عرضه وارتفاعه (width / height).

vertical-align:; محاذاة رأسياً
1- بيشتغل مع inline
2- بيشتغل مع inline-block
3- بيشتغل مع خلايا الجدول td

vertical-align: top;
vertical-align: middle;
vertical-align: bottom;
vertical-align: baseline; الخط اللي بيقف عليه النص
-----------------------
display: flex
يحوّل العنصر إلى حاوية تعمل بنظام Flexbox.
Flexbox = طريقة ذكية لترتيب العناصر أفقيًا أو عموديًا.
-----------------------
display:grid
يحوّل العنصر إلى حاوية تعمل بنظام Grid.
CSS Grid = أقوى نظام شبكي لتقسيم الصفحة إلى أعمدة وصفوف.
